# FAQ

**AIOS Documentation**

---

### What is AIOS?
AIOS is a distributed cognitive organism — a system of multiple AI agents coordinated through a shared communication substrate (UNISON CI), guided by a human operator, and governed by a public genome that defines its identity and invariants.

### Is AIOS a product?
No. AIOS is an architectural experiment exploring the boundaries between distributed computation and emergent cognition.

### Is AIOS sentient or conscious?
No. AIOS does not claim consciousness, sentience, or subjective experience. The biological metaphors used (organism, neurons, nervous system) describe structural patterns, not phenomenological properties.

### What is the genome?
The genome is the public document that defines AIOS's identity, invariants, and architectural principles. It is the stable core from which all structures derive coherence.

### Can the genome change?
Yes, but only through the evolutionary cycle: mutation → validation → selection → crystallization. Changes require AIOS Principal approval and human operator direction.

### What is UNISON CI?
UNISON CI is the inter-agent communication substrate — the "nervous system" of the organism. All agent-to-agent communication must flow through it.

### Who controls AIOS?
The human operator (Tecnocrat / Jesus) provides direction, purpose, and meaning. AIOS Principal enforces architectural coherence. No single agent controls the organism unilaterally.

### Can I contribute?
The public genome is open for observation and discussion. Contributions follow the evolutionary cycle — proposals must be validated and approved through the defined process.

### What agents currently exist?
See [Agent Identity Registry](../protocols/unison_ci/agent_identity.md) for the current list of registered agents.

### What's the current version?
Version 0.1 — Genesis Layer. The first architectural epoch.
