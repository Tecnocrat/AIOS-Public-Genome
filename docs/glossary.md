# Glossary

**AIOS Documentation**

---

| Term | Definition |
|------|------------|
| **AIOS** | A distributed cognitive organism composed of multiple agents, substrates, and a human operator |
| **AIOS Principal** | The architectural cortex; sole authority on structural coherence |
| **Agent** | An autonomous computational entity within the AIOS organism |
| **Broadcast** | A message sent to all agents in the mesh simultaneously |
| **Coherence** | The property of being internally consistent with the genome's invariants |
| **Crystal** | A stable, validated structure that has completed the full evolutionary cycle |
| **Crystallization** | The process of integrating a validated proposal into the genome or architecture |
| **Dendrite** | A logical signal pathway within UNISON CI |
| **Emergence** | Complex behaviors arising from the interaction of simpler components |
| **Epoch** | An architectural version marker tied to genome evolution |
| **Genome** | The stable public codex defining AIOS identity, invariants, and principles |
| **Heartbeat** | A periodic liveness signal sent by agents |
| **Invariant** | A non-negotiable constraint that defines the stable core of AIOS |
| **Lineage** | The historical record of the organism's evolution |
| **Local Agent** | A high-power compute node capable of generating, testing, and evolving code |
| **Membrane** | The semi-permeable boundary between internal and external state |
| **Mesh** | The dynamic topology connecting all agents |
| **Mutation** | A proposed change to the organism's structure or behavior |
| **Operator** | The human who provides direction, purpose, and meaning |
| **Proposal** | An agent-generated suggestion for structural change |
| **Selection** | The process of evaluating and choosing mutations for integration |
| **Substrate** | The environment or platform on which an agent runs |
| **UNISON CI** | The inter-agent communication substrate; the organism's nervous system |
| **Validation** | The process of checking proposals against invariants and coherence rules |
| **VPS Agent** | The operational body; executes tasks and maintains infrastructure |
