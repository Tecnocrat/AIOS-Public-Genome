### Design Principles

1. **Each tier directs the next** — the genome says "read crystal", the crystal says "read Fabric when needed"
2. **Token economy** — genome is injected every prompt (~50 lines). Crystal is read once per session. Fabric is read for architecture work.
3. **Machine-independent** — crystals are portable across substrates
4. **Self-similar** — the loading pattern itself is fractal: zoom in to load more detail
